CustomersController (/api/customers)
GET /api/customers → List all customers
GET /api/customers/{id} → Get customer by id
GET /api/customers/{id}/orders → Orders of customer
POST /api/customers → Create customer
PUT /api/customers/{id} → Update customer
DELETE /api/customers/{id} → Delete customer
POST /api/customers/bulk → Bulk insert customers
GET /api/customers/with-orders → Customers with orders
GET /api/customers/count → Total customers
GET /api/customers/full-details → Customer + Orders + OrderItems + Products


-------------------------------------------------------------------------------

OrdersController (/api/orders)
GET /api/orders → List all orders with item count
GET /api/orders/{id} → Get order with items
GET /api/orders/{id}/items → Get order items only
GET /api/orders/{id}/customer → Customer of order
POST /api/orders → Create order with/without items
POST /api/orders/{id}/items → Add item to existing order
PUT /api/orders/{id} → Update order (replace items)
DELETE /api/orders/{id} → Delete order
POST /api/orders/bulk → Bulk insert orders
GET /api/orders/with-details → Order + Customer + Items + Product
GET /api/orders/count → Total orders
GET /api/orders/total-amount → Sum(order amount)
GET /api/orders/top-customers → Customers with highest order count
